# Evidence Binder Standard (ITGC_UAR01)

## Binder Structure Per Review Period

Recommended folder pattern:

ITGC_UAR01/  
  FY2026/  
    Q1/  
      0. Index and Summary/  
      1. Access Listings/  
      2. Access Review Tracker/  
      3. Owner Certifications/  
      4. Remediation Evidence/  

## Naming Convention

Format:

`ITGC_UAR01_FY[Year]_Q[Quarter]_[Sequence]_[Description].[ext]`

Sequence mapping:

- 00 Index and summary
- 01 Access listings
- 02 Tracker
- 03 Owner certifications
- 04 Remediation evidence

Examples:

- ITGC_UAR01_FY2026_Q1_00_Index_and_Summary.docx  
- ITGC_UAR01_FY2026_Q1_01_Access_Listing_AzureEntra.xlsx  
- ITGC_UAR01_FY2026_Q1_02_Tracker_Final.xlsx  
- ITGC_UAR01_FY2026_Q1_03_AzureEntra_Owner_Certification.pdf  
- ITGC_UAR01_FY2026_Q1_04_Ticket_INC34219_Closure.png  

## Purpose

This structure allows auditors to:

- Quickly locate access listings and trackers
- Trace decisions to owner certifications
- Trace changes to remediation tickets and closure evidence
