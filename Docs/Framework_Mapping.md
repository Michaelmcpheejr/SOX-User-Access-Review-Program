# Framework Mapping

## SOX ITGC

Area:

- Logical access to programs and data that support financial reporting

How this program supports SOX ITGC:

- Defines a recurring User Access Review control (ITGC_UAR01)
- Uses application owner certification and documented remediation
- Produces structured evidence for audit re performance

## NIST SP 800 53 (Example Mappings)

- AC 2 Account Management  
  Recurring review of accounts and roles, with removal or adjustment when no longer needed.

- AC 5 Separation of Duties  
  Reviews help identify conflicting access combinations within SOX relevant processes.

- AC 6 Least Privilege  
  Privileged access is identified, justified, reviewed more frequently, and reduced or removed when excessive.

## ISO 27001 Annex A (Example Mappings)

- A.9 Access control  
  - A.9.2 User access management  
  - A.9.4 System and application access control  

- A.5 Information security policies  
  SOPs and program documentation provide policy based governance.

- A.6 Organization of information security  
  Roles and responsibilities are clearly defined for review, certification, and remediation.
