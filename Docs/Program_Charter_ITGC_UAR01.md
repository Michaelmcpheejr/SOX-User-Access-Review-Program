# Program Charter: SOX User Access Review (ITGC_UAR01)

## Purpose

Ensure all user and privileged access to SOX relevant systems is reviewed on a recurring basis, validated for appropriateness, and documented with sufficient evidence to support audit re performance and SOX ITGC compliance.

## Control Objective

Confirm that access to financial and SOX impacting systems is restricted to authorized users based on job responsibilities and that privileged access is formally reviewed and controlled.

## Frequency

- Privileged access reviews: monthly
- General user access reviews: quarterly

## Scope (Hybrid IAM Centered)

In scope:

- Azure Entra ID identity governance and role assignments
- Microsoft 365 administrative and privileged roles
- SSO integrated applications that support SOX relevant business processes
- Privileged group memberships and elevated access pathways
- Application and system owners certify access for their SOX impacting applications

Out of scope:

- Physical security
- Application code security testing
- Non SOX systems unless explicitly added to the scope inventory

## Ownership

- Execution owner: IT Compliance
- Certification owners: Application and system owners
- Remediation implementers: IAM or IT Operations
- Oversight: Internal Audit (as applicable)

## Outputs

- Completed Access Review Tracker
- Owner certifications
- Remediation tickets and closure evidence
- Evidence binder organized for audit re performance
